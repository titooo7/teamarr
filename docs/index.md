Sports EPG Generator for Dispatcharr integration.

{: .note }
Documentation is under construction.

## What is Teamarr?

Teamarr is a provider-agnostic sports data layer that generates dynamic EPG (Electronic Program Guide) entries for your sports channels. It integrates with Dispatcharr to automatically manage channel names, logos, and program data based on live sports schedules.

## Features

- **Multi-provider support** - ESPN, TSDB, Cricbuzz, HockeyTech
- **Dynamic templates** - 160+ variables for channel naming
- **Team-based & event-based EPG** - Flexible scheduling approaches
- **Dispatcharr integration** - Automatic channel lifecycle management
- **Stream matching** - Intelligent stream-to-event matching
